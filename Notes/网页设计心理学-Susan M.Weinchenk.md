# 《网页设计心理学》- Susan M.Weinschenk 读书笔记 #
------------------------------------------

## 写在前面: ##

这是一本关于网页设计方法的书，但是我并没有将这本书当做网页设计类书籍来阅读的。

书籍的英文名为《Neuro Web Design:What makes them click?》。显而易见，书籍的内容主要探讨的是网页设计中的常见的诱导用户行为的方法。我无意于网站或者互联网产品中“欺骗”用户的小把戏，但是其背后的关于人性和心理学的见解却十分受用。
